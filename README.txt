###morse_lib.h###

Simple Arduino library for a Morse code beeper.
Contains the function "morse_beep()" that reads a character x and translates it to morse code that is then played by an 8 ohm speaker.
Morse code is ITU specific.

###morse_test.ino###

Very simple Arduino sketch that utilizes the Morse library.
Serial communication provides characters that the function morse_beep() reads.


###hellomorse.mp3###
Audio file with the phrase "hello world" translated to morse.
